# Library Manager

A console-based library management system built in C#.

## About

A fully functional catalogue system for managing a book collection.
Search across multiple fields, add new books with validated input,
and delete entries by ID — all through a clean keyboard-driven menu.

## Features

- 📖 View the full book catalogue
- 🔍 Search by title, author, genre, year, or shelf location
- ➕ Add new books with input validation on all fields
- 🗑️ Delete books by ID with a confirmation prompt
- 🔢 Auto-incrementing ID system for each book entry
